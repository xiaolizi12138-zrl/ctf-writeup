# 弹窗干扰与HTML实体解码

## 题目信息
| 项目 | 内容 |
|------|------|
| 类型 | Web |
| 考点 | JavaScript弹窗干扰、HTML源码查看、HTML实体解码 |
| 难度 | ⭐ |

## 题目描述
打开网站后，无限弹出两个 alert：
- "flag就在这里"
- "来找找吧"

页面无法正常浏览。

## 解题过程

### 第一步：解决弹窗干扰

在浏览器中按 F12 打开开发者工具，切换到 Console（控制台），输入以下代码并回车：

```javascript
window.alert = function(){};
window.confirm = function(){};
window.prompt = function(){};

第二步：查看页面源码

按 Ctrl+U 查看页面源代码，发现 HTML 注释中有一段编码：

html
<!-- &#102;&#108;&#97;&#103;&#123;&#49;&#102;&#54;&#50;&#49;&#51;&#97;&#50;&#98;&#55;&#53;&#54;&#50;&#98;&#48;&#102;&#56;&#57;&#52;&#55;&#51;&#55;&#53;&#56;&#100;&#99;&#101;&#50;&#50;&#102;&#49;&#49;&#125; -->
第三步：解码
&#xxx; 是 HTML 数字实体编码，对应 ASCII 字符。手动解码或使用在线工具，得到：

text
flag{1f6213a2b7562b0f89473758dce22f11}
